## Описание

**TelegramInfoBot** — корпоративный Telegram-бот для быстрого получения информации о сотрудниках компании из независимой Excel-таблицы.

Бот подходит для компаний, где нужно быстро находить сотрудников по фамилии, отделу или ФИО, получать контактные данные, отправлять поздравления с днём рождения и ограничивать доступ только разрешённым группам, каналам и сотрудникам.

Данные берутся из Excel-файла, который может быть размещён:

- на Google Drive / Google Sheets;
- на любом файловом хостинге;
- по прямой ссылке на `.xlsx`.

## Возможности

### Для пользователей

- Поиск сотрудника по фамилии или ФИО.
- Просмотр сотрудников по отделам.
- Красивые карточки сотрудников.
- Получение телефона, email, должности, отдела и компании.
- Работа в личном чате с ботом.
- Работа в разрешённых группах и темах.
- Автоматическое удаление временных сообщений.

### Для администраторов

- Админ-панель только в личном чате с ботом.
- Управление текстом приветствия.
- Управление Excel-ссылкой.
- Управление администраторами.
- Управление whitelist групп и каналов.
- Управление разрешёнными темами групп.
- Управление отдельным чатом/темой для поздравлений.
- Перезагрузка бота.
- Диагностика и service-управление.

### Для групп и каналов

- Whitelist-доступ.
- Автоматический выход из неразрешённых групп.
- Уведомление админам при добавлении или удалении бота.
- Поддержка Telegram Topics.
- Работа только в разрешённой теме.
- Игнорирование сообщений в чужих темах.
- Защита от случайного удаления сообщений в неразрешённых темах.

## Установка

Распакуйте архив:

```bash
unzip /opt/TelegramInfoBot.zip
cd /opt/TelegramInfoBot
```

Запустите установщик:

```bash
sudo bash scripts/install.sh
```

Установщик запросит:

- Telegram Bot Token;
- ID главного администратора;
- ссылку на Excel-файл;
- информацию о компании;
- whitelist групп/каналов;
- разрешённую тему;
- чат/тему для поздравлений.

---

## Удаление

```bash
sudo bash /opt/TelegramInfoBot/scripts/uninstall.sh
```

## Переменные окружения

Пример `.env.example`:

```env
BOT_TOKEN=123456:ABCDEF
ADMIN_ID=123456789
ADMIN_IDS=123456789
EXCEL_URL=https://example.com/employees.xlsx
COMPANY_INFO=Краткая информация о компании
TIMEZONE=Asia/Yekaterinburg
AUTO_DELETE_SECONDS=30
```

Файл `.env` нельзя публиковать на GitHub.

## Whitelist групп и каналов

Бот работает только в разрешённых группах и каналах.

Если бота добавили в неразрешённый чат:

1. администраторы получают уведомление;
2. бот ждёт 30 секунд;
3. бот покидает чат.

---

## Поддержка тем Telegram

Для групп с темами можно указывать конкретную разрешённую тему:

```text
-1001234567890 1715
```

Где:

- `-1001234567890` — ID группы;
- `1715` — ID темы.

В остальных темах бот ничего не отвечает и ничего не удаляет.

---

## День рождения сотрудников

Каждый день в `09:00` по часовому поясу `Asia/Yekaterinburg` бот проверяет колонку **Дата рождения** и отправляет поздравление.

Поздравления можно направлять в отдельный чат или конкретную тему.

---

## Автоудаление сообщений

Удаляются через заданное время:

- сообщения пользователей;
- временные ответы бота;
- результаты поиска;
- промежуточные уведомления.

Не удаляется:

- главное приветственное сообщение `/start`.

---

## aiogram_dialog

В стабильной версии используется `aiogram_dialog` для:

- приветственного меню;
- кнопочного интерфейса;
- возврата на стартовое меню;
- idle reset через 15 секунд.

Если пользователь не взаимодействует с меню 15 секунд, диалог возвращается в начальное состояние.
